# TaskBarHero攻略DB

TaskBarHeroの装飾素材・彫刻素材・刻印・碑文・巻物・装備強化メモを検索できる単一HTMLです。

## 公開ページ

GitHub Pagesを有効化後、以下で表示できます。

```text
https://poceye.github.io/PageTest_repo/
```

## 注意

- `robots.txt` と HTML の `noindex,nofollow,noarchive` を設定しています。
- ただし、公開リポジトリとGitHub Pagesである以上、URLを知っている人は閲覧できます。
- 検索エンジンへの非掲載は依頼であり、完全なアクセス制限ではありません。

## データ方針

- 個別素材の主データ: WikiWiki JP
- 公式/海外ソース: Steam公式、Steam Community Guide、Steam公式ニュースなどの運用補足
- 個別装備名リスト: 高信頼な完全表がないため、現時点では装備システムカード中心
